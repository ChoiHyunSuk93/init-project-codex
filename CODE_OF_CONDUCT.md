# Code of Conduct

This project aims to be a respectful, constructive place to collaborate.

## Expected Behavior

- Be respectful and patient.
- Assume good intent.
- Give useful, specific feedback.
- Keep discussion focused on the work.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Bad-faith arguments or deliberately disruptive behavior.
- Publishing private information without permission.

The maintainer may moderate discussions, issues, and pull requests to keep the project constructive and usable for everyone.
