# 언어 정책 규칙

## 목적

사람이 읽는 생성 문서, 저장소 전용 `AGENTS.md` 오버레이, 안정적인 filename/path 규칙에 대한 authoritative 언어 정책을 정의한다.

## 선택 언어

- 사람이 읽는 생성 문서는 선택된 언어를 따른다.
- 저장소 전용 root `AGENTS.md` 오버레이도 선택된 언어를 따른다.
- 후속 clarification 질문도 선택된 언어를 따른다.

`.codex/`와 `.omx/` 아래의 OMX-managed prompt, skill, agent, config, runtime state는 OMX가 관리하며, 이 skill이 현지화하려고 다시 쓰지 않는다.
