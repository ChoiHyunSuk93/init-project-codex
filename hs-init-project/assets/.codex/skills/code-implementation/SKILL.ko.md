# Code Implementation

편집 전에 `rule/index.md`, `rule/rules/instruction-model.md`, `rule/rules/subagent-orchestration.md`, `rule/rules/cycle-document-contract.md`, `rule/rules/language-policy.md`를 읽는다.

이 skill은 구현 작업에 사용한다.

- 승인된 변경 계획 기준으로 코드와 설정을 수정한다.
- 변경 범위를 좁게 유지하고, 필요한 집중 검증을 함께 수행한다.
- 사용자-facing 문서와 working record를 섞지 않는다.
- plan을 다시 정의하지 않고, 구현 근거는 관련 rule과 cycle 문서를 참조형으로 남긴다.
