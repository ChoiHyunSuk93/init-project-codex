# Quality Review

평가 전에 `rule/index.md`, `rule/rules/subagent-orchestration.md`, `rule/rules/testing-standards.md`, `rule/rules/language-policy.md`를 읽는다.

이 skill은 품질 검토와 acceptance 검증 작업에 사용한다.

- 구현 결과를 acceptance criteria와 기대 동작에 대조한다.
- 회귀 위험과 남은 공백을 구분해 기록한다.
- 제품 파일을 직접 수정하지 않고 관찰과 판단 근거를 남긴다.
- 검토 결과는 관련 rule과 검증 기록을 참조형으로 연결한다.
