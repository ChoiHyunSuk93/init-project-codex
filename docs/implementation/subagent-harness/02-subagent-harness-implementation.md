# Subagent Harness Implementation Record

## Summary

The current repository keeps the implementation records in concern-based categories inside `docs/implementation/` and documents the same cycle model: planner, generator, and evaluator repeat until the plan passes.

## Key Points

- `docs/implementation/` stays category-based and user-facing.
- There is no `briefings/` top-level directory.
- Each category entry is a short, readable implementation record (not a separate `briefings` branch).
- The same rules are reflected in the subagent orchestration documents and working-doc guidance.
